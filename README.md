Track what models are dropping and how they're being measured.

|---|---|---|

## How to use this list

**To stay current passively** — watch `dair-ai/ML-Papers-of-the-Week` and bookmark HuggingFace Papers. Check in weekly.

**To track model progress** — watch releases on `huggingface/transformers` and follow the Ollama model library. New open-weight models show up here within days of being published.

**To understand where tooling is going** — star `langchain-ai/langchain` and `microsoft/autogen` and skim their changelogs monthly.

**To go deep on foundations** — work through `karpathy/nn-zero-to-hero` first, then `fastai/fastai`. These don't go stale.

**To track agents specifically** — `e2b-dev/awesome-ai-agents` is the best live list right now.
